My name is William Drell, and I'm a fan of Mongoose Traveller First Edition.

I created this program as a personal programming challenge while learning Python.

Also, I just wanted to make Traveller characters because I think its fun.

The primary executable will be main.py.  It will draw from config.py, careers.py, and planetgen.py.

careers.py is feature complete.  The other modules are works in progress as of 3-18-26.

rich_creator.py is partially AI generated, and I am keeping it around for reference on how to use the rich Python module.

Anyone wanting to try out this program is quite welcome to, and you are free to use it as a jumpstart
for any of your own projects.
